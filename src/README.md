# break-glb

A React component for rendering 3D GLB models with animated exploded view functionality.

See the [main documentation](../README.md) for full usage details.

## Quick Start

```tsx
import { BreakGLB } from 'break-glb'

function App() {
  return <BreakGLB modelUrl="/path/to/model.glb" />
}
```

## Installation

```bash
npm install break-glb three @react-three/fiber @react-three/drei
```

For complete API documentation, examples, and configuration options, visit:
https://github.com/ltastefinger/break-glb
